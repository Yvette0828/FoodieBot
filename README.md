# NTPU Foodie

一個為北大特區設計的美食資訊整合平台，包含餐廳爬蟲和資料庫系統。

## 功能特色

- 自動爬取北大特區餐廳資訊
- 提供餐廳名稱、地址等基本資訊
- Django 後端管理系統
- 資料庫整合
